# useScript

加载外部script

---

## API

```jsx
const [
  ready: bool
  status: 'idle' | 'loading' | 'ready' | 'error'
] = useScript(
  src: string,
  attrs: {},
);
```

### ready
是否加载完成

### status
加载状态 'idle' | 'loading' | 'ready' | 'error'

### src
js文件路径

### attrs
配置 `script` 标签的 `Attribute` 属性

---
