# useScroll #

获取元素的滚动坐标。

<hr />

## API ##
```jsx
const {
  x: Number, // x坐标
  y: Number // y坐标
} = useScroll(
  target: () => HTMLElement | MutableRefObject<HTMLElement> | Document | Window, // DOM 节点或者 Ref 对象	
);
```
